# 🎬 Online Movie Ticket Booking System

A full-stack web application that allows users to browse movies, select seats, and book tickets online.

## 🚀 Features
- User Registration & Login
- Movie Listings
- Dynamic Seat Selection
- Ticket Booking
- Payment Page
- Admin Panel

## 🛠 Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: PHP  
Database: MySQL  
Server: XAMPP

## 📂 Project Structure
admin/ – Admin dashboard  
css/ – Stylesheets  
js/ – JavaScript files  
images/ – Images used in website  

## ⚙️ Setup Instructions
1. Install XAMPP
2. Place project inside htdocs
3. Import database.sql into phpMyAdmin
4. Run localhost/movie-ticket-booking-system
